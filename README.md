### 쿼리 제출 양식
> 모든 멤버는 아래 규칙에 따라 SQL 쿼리를 제출합니다.

1. Repository Clone 또는 Pull로 최신 상태 유지
2. (1)문항에 해당하는 플랫폼, (2)문제 카테고리 디렉토리에 맞추어 쿼리 업로드
     - **공통 문제 쿼리 제출**: `Daily_SQL/{출처}/{문제_카테고리}/{문제명}/이름.sql`
     - **개별 문제 쿼리 제출**: `Daily_SQL/Archive/Season_s/Week_N/이름_problem_k.sql`
3. Commit Message: 아래 양식으로 제출
```
Submit: 이름

- 고난: 어떤 고난이 있었는지
- 해결: 어떻게 해결했는지
```

  - 리뷰어의 요청에 의한 수정의 경우: `Style: #이슈번호`
  - 개인적인 수정의 경우: `Update: 수정 사항`
  - 자세한 커밋 메시지 컨벤션은 [Notion 참고](https://www.notion.so/SQL-acf9c7bf40a741d3b3b72a1948211162?source=copy_link#27c0de90854f8070850bc9961af20236)
4. Main 브랜치로 Push
